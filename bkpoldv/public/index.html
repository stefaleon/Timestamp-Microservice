Timestamp Microservice

  1. Pass a string as a parameter, and it will be checked to see whether it contains either a Unix timestamp or a natural language date (example: January 1, 2016).
  2. If it does, both the Unix timestamp and the natural language form of that date are returned.
  3. If it does not contain a date or Unix timestamp, null is returned for those properties.
  • Example usage:
    https://timestamp-microservice-ls.herokuapp.com/December 15 2015
    https://timestamp-microservice-ls.herokuapp.com/1450137600
  • Example output:
    { "unix": 1450137600, "natural": "December 15, 2015" }
